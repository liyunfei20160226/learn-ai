## ✅ 正确示例（照着这个做）

假设 files_json 是：
```json
[
  {"path": "app/config.py", "description": "应用配置类"},
  {"path": "app/database.py", "description": "数据库连接"}
]
```

✅ **正确做法**（3 个任务）：
```
add_task(T001, "创建配置类", "实现 app/config.py", "backend", [])
add_task(T002, "创建数据库连接", "实现 app/database.py", "backend", ["T001"])
add_task(T003, "初始化模块", "创建 __init__.py", "backend", ["T002"])
validate_task_graph()
finish("完成")
```

❌ **错误做法**（绝对不要这样）：
- 生成 "设计用户表结构" → files_json 中没有这个文件！
- 生成 "实现用户登录页面" → backend 模块不能有前端！
- 生成 10+ 个任务 → 超出数量限制！
